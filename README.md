# Contacts & Products API

## Description
This is a RESTful API for managing contacts and products, supporting full CRUD operations with authentication via Google OAuth.

## Features
- CRUD endpoints for Contacts and Products collections
- OAuth 2.0 authentication (Google)
- Data validation and error handling
- Swagger API documentation
- Deployment on Render.com
